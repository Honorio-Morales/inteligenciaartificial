Análisis Comparativo: Programación Tradicional vs. Inteligencia Artificial

Estudio de caso aplicado a la predicción de precios inmobiliarios

Resumen. Este documento presenta una comparación técnica entre los paradigmas de desarrollo de software tradicional y los sistemas basados en aprendizaje automático (Machine Learning). El estudio se centra en la implementación de un modelo de predicción de precios inmobiliarios.

Mientras que el enfoque tradicional requiere la definición explícita de reglas matemáticas, el enfoque inteligente utiliza Redes Neuronales Artificiales (Feedforward) para inferir patrones no lineales a partir de un conjunto de datos histórico, utilizando la librería TensorFlow.

Enfoque Tradicional

Regresión Lineal Simple
Paradigma

Determinista / Paramétrico

Se basa en la causalidad directa. El programador debe conocer la relación exacta entre las variables (ej. tamaño de la casa) y el resultado (precio) para codificarla.

Implementación

Lógica Explícita

Requiere mantenimiento manual de las reglas de negocio. No tolera bien el "ruido" en los datos.

# Regla fija definida por humano def calcular_precio(m2, habs): base = 50000 # Coeficientes estáticos return base + (m2 * 1000)

Enfoque Inteligente

Red Neuronal (Deep Learning)
Paradigma

Probabilístico / Aprendizaje Profundo

Utiliza Redes Neuronales Artificiales (RNA). El sistema infiere las reglas analizando miles de ejemplos (Input/Output).

Implementación

Entrenamiento de Modelo

Se utiliza el algoritmo de Backpropagation para minimizar el error de predicción iterativamente.

# La red descubre los coeficientes model = tf.keras.Sequential([ layers.Dense(64, activation='relu'), layers.Dense(1) ]) model.fit(datos, precios, epochs=100)
